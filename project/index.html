Descriptions

  • Constructed a job-hunting social platform for users to display their work experiences, resumes and discuss topics.
  • Renderred the front-end pages on the server side with React.js and Next.js for Search Engine Optimization.
  • Realized 20+ GraphQL APIs with Apollo to perform CRUD operations. Implemented back end in Node.js and Express to handle requests and launched database on MongoDB atlas.
  • Solved user authentication and authorization with Passport.js.
